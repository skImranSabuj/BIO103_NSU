## 2) Cell Cycle Overview

| Phase       | English Explanation                                                          | বাংলা ব্যাখ্যা                                                      |
| ----------- | ---------------------------------------------------------------------------- | ------------------------------------------------------------------- |
| G1 phase    | Cell grows, performs normal metabolism, and prepares for DNA synthesis.      | কোষ বৃদ্ধি পায়, স্বাভাবিক বিপাক চালায় এবং DNA তৈরির প্রস্তুতি নেয়।  |
| S phase     | DNA replication occurs, producing two identical copies of genetic material.  | DNA অনুলিপি ঘটে এবং জিনগত উপাদানের দুটি অভিন্ন কপি তৈরি হয়।         |
| G2 phase    | Cell grows further and prepares proteins and structures needed for division. | কোষ আরও বৃদ্ধি পায় এবং বিভাজনের জন্য দরকারি প্রোটিন ও গঠন তৈরি করে। |
| M phase     | Nuclear division occurs through mitosis.                                     | M পর্যায়ে নিউক্লিয়াসের বিভাজন বা মাইটোসিস ঘটে।                      |
| Cytokinesis | Cytoplasm divides and two daughter cells are formed.                         | সাইটোপ্লাজম বিভক্ত হয়ে দুটি অপত্য কোষ তৈরি হয়।                      |

### Quick Formula / Shortcut

- Cell cycle order: G1 -> S -> G2 -> M -> C
- Interphase = G1 + S + G2
- Memory trick: Grow, Synthesize, Grow again, Mitosis, Cut

## 3) Mitosis

| ----------- | ------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------- |
| Prophase    | Chromatin condenses into visible chromosomes; nuclear membrane breaks down; spindle forms. | ক্রোমাটিন দৃশ্যমান ক্রোমোজোমে পরিণত হয়; নিউক্লিয়ার ঝিল্লি ভেঙে যায়; স্পিন্ডল তৈরি হয়। |

## 4) Meiosis

| Feature             | English                                                                                   | বাংলা                                                                              |
| ------------------- | ----------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------- |
| Definition          | Meiosis is a reduction division that produces haploid gametes from a diploid parent cell. | মিয়োসিস হলো হ্রাসমূলক বিভাজন যা ডিপ্লয়েড জনক কোষ থেকে হ্যাপ্লয়েড গ্যামেট তৈরি করে। |
| Number of divisions | Two divisions occur: Meiosis I and Meiosis II.                                            | দুটি বিভাজন ঘটে: মিয়োসিস I ও মিয়োসিস II।                                           |
| Result              | Four daughter cells are produced.                                                         | চারটি অপত্য কোষ তৈরি হয়।                                                           |
| Chromosome number   | Chromosome number becomes half of the parent cell.                                        | ক্রোমোজোম সংখ্যা জনক কোষের অর্ধেক হয়ে যায়।                                         |
| Role                | Essential for sexual reproduction and variation.                                          | যৌন জনন ও বৈচিত্র্যের জন্য অপরিহার্য।                                              |

### Mitosis vs Meiosis

| Feature             | Mitosis           | Meiosis          |
| ------------------- | ----------------- | ---------------- |
| Number of divisions | 1                 | 2                |
| Daughter cells      | 2                 | 4                |
| Chromosome number   | Same as parent    | Half of parent   |
| Genetic similarity  | Identical         | Non-identical    |
| Main purpose        | Growth and repair | Gamete formation |

## 6) Side Notes, Shortcuts, and Common Mistakes

### High-Yield Tips

- DNA replication always happens before division, mainly in S phase.
- Mitosis keeps chromosome number constant; meiosis reduces it by half.
- PMAT is the standard mitosis sequence: Prophase, Metaphase, Anaphase, Telophase.
- Binary fission is not mitosis because prokaryotes have no true nucleus.

### Common Mistakes to Avoid

- Mistake: "DNA replication happens during mitosis." Correct: It occurs during S phase of interphase.
- Mistake: "Mitosis produces four cells." Correct: Mitosis produces two daughter cells.
- Mistake: "All cell division reduces chromosome number." Correct: Only meiosis reduces chromosome number.
- Mistake: "Binary fission and mitosis are the same." Correct: They are different processes in different cell types.

## 7) Solved Examples

### Example 1: Why is DNA replication necessary before cell division?

- Each daughter cell needs a full set of instructions to survive.
- Without replication, one or both daughter cells would receive incomplete genetic material.

Final answer: DNA replication ensures both daughter cells inherit complete and identical genetic information before division.

### Example 2: A diploid cell with 46 chromosomes undergoes meiosis. How many chromosomes will each daughter cell have?

- Diploid number = 46
- Meiosis reduces chromosome number by half
- 46 / 2 = 23

Final answer: Each daughter cell will have 23 chromosomes.

## 8) MCQ Practice (Lec 13-14)

1. DNA replication occurs mainly in:
   A) G1 phase B) S phase C) M phase D) Cytokinesis  
   Answer: B  
   Why: S phase is the synthesis phase where DNA is copied.

2. The stage of mitosis where chromosomes line up in the middle is:
   A) Prophase B) Anaphase C) Metaphase D) Telophase  
   Answer: C  
   Why: In metaphase, chromosomes align at the metaphase plate.

3. Mitosis produces:
   A) Four haploid cells B) Two identical diploid cells C) Two haploid cells D) Four identical cells  
   Answer: B  
   Why: Mitosis creates two genetically identical daughter cells.

4. Which process reduces chromosome number by half?
   A) Binary fission B) Cytokinesis C) Mitosis D) Meiosis  
   Answer: D  
   Why: Meiosis is reduction division.

5. Binary fission is common in:
   A) Human cells B) Bacteria C) Flowering plants D) Fungi only  
   Answer: B  
   Why: Prokaryotes such as bacteria reproduce mainly by binary fission.

6. Sister chromatids separate during:
   A) Metaphase B) Prophase C) Anaphase D) Interphase  
   Answer: C  
   Why: The spindle pulls sister chromatids apart in anaphase.

7. Which of the following is an asexual reproduction method?
   A) Fertilization B) Meiosis C) Budding D) Gamete fusion  
   Answer: C  
   Why: Budding is an asexual reproduction process.

8. In humans, meiosis produces cells with how many chromosomes?
   A) 46 B) 23 pairs C) 23 D) 92  
   Answer: C  
   Why: Human gametes are haploid and contain 23 chromosomes.
